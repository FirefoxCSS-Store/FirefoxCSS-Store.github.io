Lectura para el equipo

  • El coste de arranque baja mucho porque AGENTS.md ya no carga toda la historia.
  • La memoria total crece, pero vive en archivos Markdown consultables por tema.
  • Cuando se necesita más contexto, Basic Memory recupera notas enlazadas en lugar de meterlo todo siempre.

Cambio de modelo mental

Antes

Un archivo monolítico

  • AGENTS.md apuntaba a MEMORY.md.
  • La memoria completa entraba al contexto al inicio.
  • Más coste aunque la tarea no necesitara todo.
→

Ahora

Grafo de notas atómicas

  • AGENTS.md apunta al proyecto Basic Memory del repo.
  • Las notas viven en basic-memory/ y se versionan.
  • El contexto se recupera por relevancia y enlaces.
Resultado: casi 4 veces más conocimiento documentado, pero normalmente solo se cargan entre 399 y 913 tokens al empezar.
