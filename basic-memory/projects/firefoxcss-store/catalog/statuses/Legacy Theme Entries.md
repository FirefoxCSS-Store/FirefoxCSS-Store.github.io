# Legacy Theme Entries

Legacy theme entries are existing catalog entries where the original submitter relationship is unknown. They are marked with `submitterRole: "legacy"` to preserve that distinction.

Legacy entries still belong to [[Theme Catalog Source of Truth]] and should follow the same validation and status rules as other entries. Their legacy marker is important context for contribution and ownership decisions.

## Observations
- [status] Legacy entries represent pre-existing catalog records with unknown original submitter relationship #legacy
- [metadata] Legacy entries use `submitterRole: "legacy"` #legacy
- [ownership_context] The legacy marker prevents implying a known submitter relationship #legacy
- [validation] Legacy entries should still follow theme schema validation #validation
